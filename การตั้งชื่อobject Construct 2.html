#การตั้งชื่อ object Construct 2

2.กดไปที่ object ที่ต้องการตั้งชื่อ (ในกรอบสีแดงคือobject)

4.กดไปในobjectที่ตั้งชื่อ

5.เป็นอันเสร็จเรียบร้อย
